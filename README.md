# نظام صيدلية مجد لإدارة المخزون

نظام ويب عربي لإدارة مخزون الصيدليات والفروع مع دعم رفع ملفات Excel والمقارنة بين الفروع.

## المميزات
- إدارة المخزون
- رفع ملفات Excel
- مقارنة الفروع
- تنبيهات النواقص
- متابعة تواريخ الانتهاء
- لوحة تحكم عربية حديثة

## التقنيات المستخدمة
- React
- Node.js
- Express
- MySQL

## التشغيل
```bash
npm install
npm run dev
```

## المشروع
تم تطوير النظام لإدارة عدة فروع صيدليات عبر الإنترنت مع تحديث المخزون بشكل تلقائي.
